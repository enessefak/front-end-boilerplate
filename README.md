# Gulp 4 with Webpack 4

## How to use

Download the repo and run:

```
YARN Package Manager
yarn install
yarn run start
yarn run build

only script bundle => yarn run build:js
only styles bundle => yarn run build:css
only img bundle    => yarn run build:img
only html bundle   => yarn run build:html

NPM Package Manager
npm install
npm run start
npm run build

only script bundle => npm run build:js
only styles bundle => npm run build:css
only img bundle    => npm run build:img
only html bundle   => npm run build:html
```
